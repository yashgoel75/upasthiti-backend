# Upasthiti Backend

API Routes

- api/admin            - Get Admin Details
- api/admin/update     - Update Admin Profile 
- api/admin/faculties/upload  - Admin Add Faculty Data (in CSV)
- api/admin/students/upload   - Admin Add Student Data (in CSV)
- api/count            - Get Stats
- api/faculty          - Get Faculty Details
- api/student          - Get Student Details

`Server PORT is handled using PORT environment variable. by default, it is 8000`
